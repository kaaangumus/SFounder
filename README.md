# SFounder

A small security utility for checking web-accessible paths that may indicate exposed shells or suspicious files during authorized assessments.

## Status

Legacy utility. Kept as a small research/tooling project.

## Usage

Review the script options before running it against any target. Use it only in a lab, on your own assets, or within a clearly authorized security test.

```bash
python3 main.py
```

## Notes

- Keep target scope explicit.
- Validate findings manually before reporting.
- Do not scan third-party systems without permission.

## Legal Notice

This project is intended for education, research, and authorized security testing only.
